# 📱 ICORE | Premium Tech Storefront

## 🎯 Core Philosophy

Generic marketplaces often bury technical specifications, leading to buyer hesitation for high-cost hardware. **ICORE** eliminates "Information Asymmetry" by treating technical specs (SoC, RAM, Storage) as a primary UI element, creating a white-glove shopping experience for tech-savvy consumers.

---

## ✨ Key Capabilities

### 🖥️ Technical Spec Rendering
* **Data-Dense UI:** Custom components designed to display complex device variants and deep-dive hardware specifications at a glance.
* **Apple-esque Aesthetics:** A minimalist design language focused on product photography and clean typography.

### 💳 Secure Transaction Flow
* **Optimized Checkout:** A high-conversion pipeline integrated with **Stripe** for secure, premium-tier transactions.
* **Direct fulfillment:** Order routing logic designed to move high-value assets quickly into the admin fulfillment queue.

### 🛡️ Scalable Infrastructure
* **Type-Safe Database:** Leveraging **Drizzle ORM** and **PostgreSQL** for a rigid yet flexible technical inventory schema.
* **Modern State Management:** Ultra-fast, lightweight client-side state handling via **Zustand**.

---

## 🛠️ Tech Stack

### Frontend & Logic
* **Next.js:** Full-stack React framework for SSR and optimized performance.
* **Zustand:** Minimalist, fast state management for cart and variant logic.
* **Shadcn UI & Tailwind:** Professional-grade design system and utility-first styling.
* **Auth.js:** Secure, flexible authentication for premium user accounts.

### Backend & Storage
* **PostgreSQL:** Robust relational database for complex product relationships.
* **Drizzle ORM:** Type-safe database interactions and rapid migrations.
* **Cloudinary:** High-fidelity image hosting for 4K tech asset rendering.
* **Stripe:** Industry-standard secure payment processing.
